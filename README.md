# 🚀 AI Fraud Detection Platform

An advanced AI-powered analytics platform built using Streamlit and Machine Learning.

## ✨ Features

* Upload any CSV dataset
* Automatic dataset preprocessing
* Dynamic AI model training
* Real-time predictions
* Fraud detection analytics
* Correlation heatmaps
* Feature importance visualization
* Random sample generator
* PDF report generation
* Supports numeric + categorical datasets

## 🛠 Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* Plotly
* ReportLab

## 📊 Project Workflow

1. Upload dataset
2. Select target column
3. Generate random dataset values
4. Train AI model
5. Predict outcomes
6. Download analytics reports
